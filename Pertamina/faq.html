Berikut ini adalah hal-hal yang sering dipertanyakan mengenai Karir PT. Pertamina (Persero)

Anda hanya diperkenankan melamar untuk 1 kali saja.

Kami menyarankan peserta untuk menggunakan komputer yang tersambung dengan koneksi internet broadband yang stabil. Hal ini dilakukan untuk menghindari kerusakan data selama proses pendaftaran.

Siapkan komputer atau laptop yang telah terinstall web browser minimal Internet Explorer 10 atau yang terbaru atau Mozilla Firefox minimal versi 3.0 atau Google Chrome minimal versi 16. Lengkapi komputer atau laptop Anda dengan mouse. Pastikan web browser tersebut dapat menerima cookies dan pastikan komputer atau laptop Anda tersambung dengan jaringan internet yang stabil.

Anda diminta untuk mengupload dokumen yaitu KTP, foto, ijazah, transkrip nilai, CV, dan dokumen sertifikasi lainnya.

Untuk kelancaran pendaftaran Anda, kami menyarankan Anda menggunakan minimal Internet Explorer 10 atau versi terbaru atau Mozilla Firefox minimal versi 3.0 atau versi terbaru atau web browser Google Chrome minimal versi 16 atau versi terbaru untuk melaksanakan Aplikasi secara online. Serta mengeset komputer Anda untuk dapat menerima cookies.

Aplikasi bersifat online, dan dapat dilakukan dimana saja selama web browser yang direkomendasikan telah terinstal pada komputer/laptop dan terhubung dengan internet.

Cookies adalah bagian kecil dari data yang dikirimkan dari sebuah situs web dan disimpan dalam web browser pengguna ketika pengguna melakukan browsing di situs web tersebut. Setiap kali pengguna membuka website, web browser mengirimkan cookie kembali ke server untuk memberitahu website aktivitas pengguna sebelumnya.

Internet Explorer:
Pilih menu Tools ⇒ Pilih Internet Options ⇒ Pilih General ⇒ Pilih Browsing History ⇒ Klik Delete.. ⇒ Klik Delete cookies.. ⇒ Klik OK.
Mozilla Firefox:
Pilih menu Tools ⇒ Pilih Clear Recent History (Ctrl+Shift+Delete) ⇒ Pilih time range to clear, pilih everything ⇒ pada detail, pilih check all, kemudian klik tombol “clear now”.

JavaScript merupakan bahasa pemrograman yang berkerja di sisi klien sebagai bagian dari web browser. Implementasi javascript memungkinkan script sisi klien untuk berinteraksi dan mengontrol web browser, berkomunikasi asynchronous, dan mengubah tampilan.
